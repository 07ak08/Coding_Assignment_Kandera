# Coding_Assignment_Kandera

How to reproduce this work:

1. download the airnbnb_london_listing dataset from https://osf.io/qyca8/overview
2. create a respository in GitHub

reproducing the python code:

1. save the airbnb_london_listing dataset in the folder you are working in
2. run the first py file (cleans the data)
3. run the second py file (prepares the data for analysis)

4. open git bash and commit all the python related files, then push to repository


reproducing the stata code:

1. save the airbnb_london_listing dataset in the folder you are working in
2. run the first do file (demonstrates folder structure)
4. run the second do file (cleans the data)
5. run the third do file (prepares the data for analysis)
6. run the fourth do file (does summary statistics)
7. run the last do file (generates a graph)

8. open git bash and commit all the stata related files, then push to repository
